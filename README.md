# Start

`vscode` 에서 실행 및 디버그로 바로 실행하세요! server, client side 모두 vscode 디버깅 사용 시 자동으로 실행하고 중지하도록 구성되어 있습니다! 😎

> 단, 컨테이너를 프로비저닝하는데 시간이 소요되므로 아래의 명령어로 1회 컨테이너 apply 이후 디버깅을 시작하는 것을 추천합니다.
>
> ```shell
> docker-compose up -d
> ```
